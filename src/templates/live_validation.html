How it works
  • Each input has hx-post="/validate/{field}", hx-trigger="blur", and hx-target pointing to its feedback div.
  • On blur, HTMX POSTs the field value to the server and swaps the feedback div with the returned HTML fragment.
  • The server returns an HX-Trigger: validationResult header; the LiveValidator JS listens for it and applies is-valid / is-invalid classes to the input.
  • The final submit still uses standard server-side Pydantic validation as a safety net.
